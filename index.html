WanderSync

A Collaborative Travel Management System

Introduction

WanderSync simplifies creating and managing travel itineraries for solo and group travel. With real-time collaboration, intuitive features, and a user-friendly design, the app ensures seamless planning, leaving you free to enjoy the journey.

Design & Architecture

The app follows a modular architecture, integrating Firebase for real-time database management and user authentication. Key design patterns include:

  • Observer Pattern: Used to dynamically update the UI when changes occur in the database.
  • Singleton Pattern: Ensures a single instance of the Firebase database reference is used throughout the app.
  • Strategy Pattern: Implements flexible layout management for the RecyclerView.
